# Shardus CLI Tool

# Features

The shardus-cli tool lets you:

  • Initialize Shardus app projects from templates.
  • Create and manage local test networks running your app.
  • Collect debug data from a running Shardus app network.

# Setup

# Pre-install

Make sure you have the following installed and configured (we recommend using nvm) to manage your Node.js and npm versions):

  • Node.js (10.x.x)
  • npm (6.x.x)
  • Git

Then, install the node-gyp dependencies for your platform listed here.

On Ubuntu, it goes something like this:

$ sudo apt update && sudo apt install python2.7 make g++
$ npm config set python python2.7

# Install

$ npm i -g shardus

# Windows users

Unfortunately, due to some features not supported by Windows, Shardus is currently only available on Linux and Mac
