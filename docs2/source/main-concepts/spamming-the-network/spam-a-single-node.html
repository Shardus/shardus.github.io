# Spam a single node on Shardus

  1. Install project dependencies

    npm install

  2. Start the network with a single node

    npm run start

  3. Create a spam command in client.js to spam the node

vorpal
  .command(
    "tokens spam create <amount> <to> <seconds> <tps>",
    "Spams the network with create token transactions"
  )
  .action(function(args, callback) {
    let spam = setInterval(() => {
      for (let i = 0; i < args.tps; i++) {
        injectTx({
          type: "create",
          from: "0".repeat(32),
          to: walletEntries[args.to],
          amount: args.amount,
          timestamp: Date.now()
        });
      }
    }, 1000);
    setTimeout(() => {
      clearInterval(spam);
      callback();
    }, 1000 * args.seconds);
  });

  1. Run the client

    node client.js

  2. Use the wallet create command to intialize a named wallet

    wallet create myWallet

  3. Run the spam command to spam the single node

    tokens spam create 100 myWallet 20 20

  4. Visit localhost:3000 in your browser to view the incoming network transactions
